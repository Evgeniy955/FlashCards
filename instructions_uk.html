Інструкція з використання

1. Початок роботи

На головному екрані натисніть кнопку "Get Started". Відкриється вікно, де ви можете обрати джерело слів:

  • Built-in: Вибрати один із готових, вбудованих словників.
  • From Computer: Завантажити власний файл зі словами у форматі Excel (.xlsx).

2. Підготовка файлів

Словник слів (.xlsx)

Для створення власного набору карток, ваш файл Excel має відповідати чіткій структурі. Для автоматичного визначення мов додайте заголовок у першому рядку:

  • Рядок 1 (Заголовок): Введіть назву мови у відповідну колонку (наприклад, "Іспанська" в A1, "Англійська" в C1). Це рекомендований спосіб.
  • Рядок 2 і далі (Слова):
    • Колонка A: Слово першою мовою.
    • Колонка B: Має бути порожньою.
    • Колонка C: Переклад другою мовою.
    • Колонка D: Має бути порожньою.

Приклад структури для кількох наборів:


|     A     | B |      C      | D |      E     | F |       G      |
|-----------|---|-------------|---|------------|---|--------------|
| Іспанська |   |  Англійська |   |  Німецька  |   |  Англійська  | <-- Рядок заголовка
|-----------|---|-------------|---|------------|---|--------------|
|   gato    |   |     cat     |   |   Katze    |   |     cat      | <-- Рядки зі словами
|   perro   |   |     dog     |   |    Hund    |   |     dog      |

Ви можете розміщувати кілька наборів слів на одному аркуші. Наступний набір починатиметься з колонки E, потім I, і так далі (з кроком у 4 колонки).

Примітка: Якщо ви завантажите файл без рядка заголовка, він все одно працюватиме, але мови будуть названі загально (наприклад, "Мова A", "Мова C").

Речення-приклади (опціонально)

Щоб бачити слова в контексті, ви можете завантажити файл із прикладами. Цей файл має містити приклади англійською мовою.

  • Формат .xlsx: Колонка A — англійське слово, Колонка B — речення-приклад.
  • Формат .json: Об'єкт, де ключ — англійське слово (в нижньому регістрі), а значення — речення.

3. Процес тренування

  • Вибір набору: Після завантаження файлу оберіть один із доступних наборів слів для початку.
  • Інтервальне повторення: Додаток показує лише ті слова, які нові або час яких настав для повторення.
  • Кнопки "Know" / "Don't know": Натискання "Know" збільшує інтервал до наступного показу слова. Натискання "Don't know" скидає прогрес.
  • Перевертання картки: Натисніть на картку, щоб побачити переклад.
  • Прослуховування: Натисніть іконку динаміка, щоб почути вимову (англійською).
  • Напрямок перекладу: Використовуйте перемикач (напр., "Іспанська ↔ Англійська"), щоб змінити напрямок тренування.

4. Керування прогресом

Весь ваш прогрес зберігається у вашому акаунті, прив'язаному до конкретного словника.

  • Learned: Відкриває вікно зі списком усіх вивчених слів.
  • Reset: Повністю видаляє весь збережений прогрес для поточного словника.
  • Change: Дозволяє змінити поточний словник.
