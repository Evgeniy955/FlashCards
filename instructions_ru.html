Инструкция по использованию

1. Начало работы

На главном экране нажмите кнопку "Get Started". Откроется окно, где вы можете выбрать источник слов:

  • Built-in: Выбрать один из готовых, встроенных словарей.
  • From Computer: Загрузить собственный файл со словами в формате Excel (.xlsx).

2. Подготовка файла Excel

Чтобы создать собственный набор карточек, ваш файл Excel должен соответствовать четкой структуре. Для автоматического определения языков добавьте заголовок в первой строке:

  • Строка 1 (Заголовок): Введите название языка в соответствующую колонку (например, "Испанский" в A1, "Английский" в C1). Это рекомендуемый способ.
  • Строка 2 и далее (Слова):
    • Колонка A: Слово на первом языке.
    • Колонка B: Должна быть пустой.
    • Колонка C: Перевод на второй язык.
    • Колонка D: Должна быть пустой.

Пример структуры:


|     A     | B |      C      | D |
|-----------|---|-------------|---|
| Испанский |   |  Английский |   | <-- Заголовок
|-----------|---|-------------|---|
|   gato    |   |     cat     |   | <-- Слово
|   perro   |   |     dog     |   |

Вы можете размещать несколько наборов слов на одном листе. Следующий набор будет начинаться с колонки E, затем I, и так далее (с шагом в 4 колонки).

Примечание: Если вы загрузите файл без строки заголовка, он все равно будет работать, но языки будут названы обобщенно (например, "Language A").

3. Процесс тренировки

  • Выбор набора: После загрузки файла выберите один из доступных наборов слов для начала.
  • Интервальное повторение: Приложение показывает только те слова, которые являются новыми или время которых пришло для повторения.
  • Кнопки "Know" / "Don't know": Нажатие "Know" увеличивает интервал до следующего показа слова. Нажатие "Don't know" сбрасывает прогресс.
  • Переворачивание карточки: Нажмите на карточку, чтобы увидеть перевод.
  • Прослушивание: Нажмите иконку динамика, чтобы услышать произношение (на английском).
  • Направление перевода: Используйте переключатель (напр., "Испанский ↔ Английский"), чтобы изменить направление тренировки.

4. Тренировка незнакомых слов

Слова, которые вы помечаете как "Don't know", сохраняются в отдельном списке. После завершения сессии появится кнопка для их тренировки. Вы можете переключаться между режимами:

  • Write: Введите перевод вручную.
  • Guess: Выберите правильный перевод из трех вариантов.

5. Управление прогрессом

Весь ваш прогресс сохраняется в вашем аккаунте, привязанном к конкретному словарю.

  • Learned: Открывает окно со списком всех выученных слов.
  • Reset: Полностью удаляет весь сохраненный прогресс для текущего словаря.
  • Change: Позволяет сменить текущий словарь.
